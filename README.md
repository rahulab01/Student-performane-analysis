# 📊 Student Performance Analysis

## 📌 Internship Task 1 – Data Analysis with Python

This project is part of my internship where I performed data analysis on student performance data using Python. The goal is to explore key factors affecting students' academic results.

---

## 🎯 Objective

* Analyze student performance data
* Identify patterns and relationships
* Understand how study time, gender, and other factors affect grades

---

## 🛠️ Tools & Technologies

* Python 🐍
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook / VS Code

---

## 📂 Dataset

The dataset contains information about students such as:

* Demographics (age, gender, address)
* Study time
* Family background
* Academic performance (G1, G2, G3)

---

## 📊 Analysis Performed

* Data cleaning (handling duplicates, checking missing values)
* Statistical analysis
* Correlation analysis
* Group-based analysis (e.g., gender comparison)

---

## 📈 Visualizations

* Histogram of final grades
* Scatter plot (study time vs performance)
* Bar chart (gender comparison)
* Heatmap (correlation between variables)

---

## 🔍 Key Insights

* Study time shows a positive correlation with final grades
* Students with higher study time tend to perform better
* Performance differences observed between genders
* Previous grades (G1, G2) strongly influence final grade (G3)

---

## 🚀 How to Run

1. Clone this repository
2. Open the notebook (`.ipynb`) in Jupyter or VS Code
3. Run all cells to see analysis and visualizations
